Better Buttons and forms

app/views/posts/index.html.erb

We start off by adding table class to our index page table section. The result is really smart.

Every tutorial I've done recently has stripped out all the table elements from the index. The body column will need truncating at some point, I'll be patient as I'm learning some really good practices.

Style the form partial. app/views/posts/_form.html.erb

  • Added form-group Bootstrap class to the divs with field class. Nothing much happens.
  • The real magic is adding the Bootstrap class form-control to the text field and text area. Suddenly the forms looks smart and are responsive.

Deep Dive: Build a Blogging Platform 6

Displaying the Date - injecting some Ruby

In app/views/posts/index.html.erb

As the posts index is displayed in a table we first have to add the title of date to the table head.

Inside erb tags we use our first bit of ruby logic. post.created_at.to_date, this adds a column to our table which will display the date a post was created.

In app/views/posts/show.html.erb

We also show the date when we click on a post to view it. To do this we add another field and inject Ruby using erb tags. @post.created_at.to_date.
